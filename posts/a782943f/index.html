Deformable_DETR

题目： Deformable DETR: Deformable Transformers for End-to-End Object Detection
链接： 2010.04159
代码： GitHub - fundamentalvision/Deformable-DETR: Deformable DETR: Deformable Transformers for End-to-End Object Detection.
来源： ICLR 2021
单位： SenseTime Research、University of Science and Technology of China、The Chinese University of Hong Kong

DETR 的缺陷

Detr 存在训练收敛慢以及小物体检测性能差的缺陷。作者认为这两点都与 transformer 本身的特性有关。训练收敛慢是因为 attention 在最开始会对所有 pixel 均匀关注，因此需要很长时间收敛到有意义的区域。小物体检测性能差是由于 transformer 的计算是平方级的，因此想要利用高分辨率的 feature map 很困难。

Method

Deformable Attention Module


结构如上图所示，与正常的 attention module 区别比较大：

  • 正常的 attention module 会利用 query 与 key 计算出 attention map，然后根据这个 attention map 对 value 进行加权，得到该 query 的输出；
  • 在 deformable attention module 里，attention map 是用 linear 层直接算出来的，关注哪些 pixel 也是用 linear 层算出来的，相当于根据算出来的 offset 在 value 里找到对应的值，然后用算出来的 attention weight 对这些值加权，得到该 query 的输出。

Multi-scale Deformable Attention Module

Deformable attention module 可以比较容易地拓展到多层，只要对一个 query 在多层特征里计算 offset，加权的时候对多层特征加权就行了。

Deformable Transformer Encoder

把 DETR 里的 encoder 换成了 deformable 版本，每一个 encoder 的输入和输出都是多尺度特征。因为这样子有不同 level 的同一个 pixel 参与计算，因此加入了一个 level embedding 用于区分不同尺度的特征。

Deformable Transformer Decoder

Decoder 里仅有 cross attention 被替换，self attention 保持了原结构。Query point 由 query embedding 通过 linear 层计算得到，相当于预测的物体中心初始值。Detection head 输出物体中心相对 query point 的偏移，以及物体的长和宽。
