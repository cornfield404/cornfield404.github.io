各种 normalization

发表于2025-08-06|更新于2025-08-11|技术积累博客笔记
|总字数:391|阅读时长:1分钟|浏览量:

说明

本篇笔记主要参考了知乎用户王几行XING 的文章 https://zhuanlan.zhihu.com/p/19150825562 以及 LayerNorm在NLP和CV的使用区别 | 哒鸽杂货间。

Batch Norm（BN）

  • 针对单个通道，在 batch 维度上计算均值和方差（跨样本单通道）
  • x^=x−μbatchσbatch2+ϵ\hat{x}=\frac{x-\mu_{batch}}{\sqrt{\sigma_{batch}^2+\epsilon}}x^=σbatch2​+ϵ​x−μbatch​​
  • 对小 batch 的训练不太适用（均值和方差的统计可能不稳定）

Layer Norm（LN）

  • 针对每个样本的所有 channel 归一化（单样本跨通道）
  • x^=x−μlayerσlayer2+ϵ\hat{x}=\frac{x-\mu_{layer}}{\sqrt{\sigma_{layer}^2+\epsilon}}x^=σlayer2​+ϵ​x−μlayer​​
  • 对小 batch 比较友好，但是忽略了 batch 内的统计信息
  • 当数据是图像的时候这个 layer norm 相当于在每张图片上计算平均值和方差，当数据是自然语言的时候是在单个 token 上计算均值和方差

Instance Norm（IN）

  • 对每个样本的每个通道归一化（单样本单通道）
  • x^=x−μinstanceσinstance2+ϵ\hat{x}=\frac{x-\mu_{instance}}{\sqrt{\sigma_{instance}^2+\epsilon}}x^=σinstance2​+ϵ​x−μinstance​​
  • 比较适合风格迁移等任务

Group Norm（GN）

  • 介于 LN 和 IN 之间，沿 batch 维度分成几组
  • x^=x−μgroupσgroup2+ϵ\hat{x}=\frac{x-\mu_{group}}{\sqrt{\sigma_{group}^2+\epsilon}}x^=σgroup2​+ϵ​x−μgroup​​

RMSNorm

  • 大模型中常用的 normalization 之一
  • Layer norm 的变种，放弃了 layer norm 里面计算均值的操作（包括减均值操作以及标准差计算里的减均值操作），提升了计算速度

\hat{x}&=\frac{x}{RMS(x)} \
RMS(x)&=\sqrt{\frac{1}{n}\sum_{i=1}^nx_i^2+\epsilon}
\end{align}
