InstructSeg

发表于2024-12-26|更新于2025-04-24|技术积累论文笔记

题目： InstructSeg: Unifying Instructed Visual Segmentation with Multi-modal Large Language Models
链接： https://arxiv.org/pdf/2412.14006
代码： https://github.com/congvvc/InstructSeg
来源： arxiv
单位： Tsinghua Shenzhen International Graduate School, Tsinghua University；Meituan Inc.

本文目的

为图像和视频的 referring segmentation 和 reasoning segmentation 设计一个统一的 end-to-end 框架。

Method

Object-aware Video Perceiver


该模块根据输入的待分割视频帧及文本信息，提取 query 特征：

frt=FCLIP(Irt)Qt=CrossAttn(Q,Concat[Fp(frt),ε])\begin{align} f_r^t&=F_{CLIP}(I_r^t) \\ Q_t&=CrossAttn(Q,Concat[F_p(f_r^t), \varepsilon]) \end{align} frt​Qt​​=FCLIP​(Irt​)=CrossAttn(Q,Concat[Fp​(frt​),ε])​​

其中，ε\varepsilonε 是文本 token，IrtI_r^tIrt​ 是每一帧的图像输入，FpF_pFp​ 指对其图像及文本空间的投影模块，QtQ_tQt​ 指每一帧的输出。

Vision-guided Multi-granularity Text Fusion


LLM 输出的 detailed text embed 通过 global average pooling 得到 global text embed EgE_gEg​。两部分 text embed 拼在一起作为 cross attention 的一个输入，图像中提取的图像特征与 global text embed 拼在一起作为 cross attention 的另一个输入，经过多轮 cross attention 与 FFN 后，生成 multi-grained text embed 作为输出。

实验结果


个人看法

提出了一个可以同时处理 video 和 image reasoning segmentation 的算法，其中融合多帧的方式是单独提取每一帧的 learnable query 并沿时间维度 concat 到一起。
